Marcos de trabajo de Java

La palabra inglesa "FRAMEWORK" (marco de trabajo) define, en términos generales, un conjunto de estandarizado de conceptos, prácticas y criterios para enfocar un tipo de problemática particular que sirve como índole similar. (herramientas y recursos para el proyecto o tecnología)

En el desarrollo de software, un framework o infraestructura digital es una estrcutura conceptual y tecnológica de soporte definidio, normalemnte con artefactos o módulos de software concretos, que puede servir de base para la organización y desarrollo de software, típicamente hablando puede incluir soporte de programas, bibliotecas, y un lenguaje interpretado, entre otras herramientas, para así desarrollar y unir componentes de un proyecto.
