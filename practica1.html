Ejemplo de Applet con Java

Un applet conforme a lo que encontramos en la página de Oracle: "es una aplicación hecha en Java que se llama desde un docuemnto HTML y se ejecuta dentro de un navegador". Es necesario recordar que el navegador debe tener instalado el plugin de la Máquina Virtual de Java (JVM).


Máquina Virtual de Java: Es un programa que se encarga de ejecuatr el bytecode generado por el compilador de Java.
